# UI Template

This is a simple web ui template created using VITE + React + TypeScript + Shadcn.

## Introduction to The UI

It is build as an SPA.

The UI include:
- A headbar showing Appname
- A sidebar to browse different pages
- Mobile responsiveness
- Darkmode support
